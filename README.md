# ENERGY-DELAY-OPTIMIZATION-FOR-A-THREE-STAGES-CMOS-BUFFER
Si ottimizza una catena di buffer CMOS tramite sensitivity analysis e simulazioni Monte Carlo. Si estrae una curva empirica nel piano Energy-Delay, confrontandola con quella ottimale di Pareto per identificare il giusto tradeoff energia-ritardo e analizzare l'impatto di parametri come tensione di alimentazione e sizing dei transistor.
